A simple banking application 
Features - Deposit money - Withdraw money - Check balance 

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Download the files
2. Open `Bank Account System.html` in your browser

## Files
- `Bank Account System.html` - Main page
- `Css.css` - Styling
- `Java script.js` - Functions
